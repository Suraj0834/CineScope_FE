## 🛠️ How to Create Visual Assets

### For Feature Graphic (1024x500):

**Tools you can use:**
1. **Canva** (Free) - https://canva.com
   - Search for "App Banner" template
   - Customize with your app colors (purple/violet theme)
   - Add app name and tagline

2. **Figma** (Free) - https://figma.com
   - Create 1024x500 frame
   - Add gradient background (#667eea to #764ba2)
   - Add app logo and text

3. **Photoshop/GIMP** (If you have it)
   - Use provided dimensions
   - Design with your branding

### For Screenshots:

**Easiest Method:**
1. Run your app on emulator
2. Take screenshots of each main screen
3. Add phone frame using:
   - https://mockuphone.com (Free)
   - https://screenshots.pro (Free)
   - https://smartmockups.com (Some free options)

**Recommended Screens:**
- Home screen with trending movies
- Movie detail page
- Watchlist page with movies
- AI chat conversation
- Profile page
